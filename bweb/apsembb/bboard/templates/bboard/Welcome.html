Welcome to my bulletin board website!

How this website was created:

To start, I installed Adobe Dreamweaver, Django, and Visual Studio Code – Insiders to help create the website. Django functioned as a framework for the programming language Python, and I used the Visual Studio Code – Insiders program to view and edit the code. Adobe Dreamweaver was used to create and edit html files to serve as templates for the code.

I viewed tutorials on how to create and format html files in Adobe Dreamweaver, and also spent time following a tutorial on how to create a project in Django. The Django tutorial helped me create a project that allowed a user to create questions for other users to vote on, as well as view the results of the polls.

After finishing this tutorial, I started to work on the bulletin board website, using the code from the tutorial project as a reference for the Django language syntax. I created the different webpages that I wanted to use for the project in Adobe Dreamweaver as html files, and then copied them into the Django project file. I then used views to call the templates and mapped them to different urls within the web site. After learning more about the Django syntax online, I was able to write the code enabling users to register accounts and login. I also programmed an html file in Django to allow for the posting of comments and the tracking of which user posted them and when. This involved creating a “Comment” model in Django, which was a basic character field that could be used in different views and was interactive with all users.

Afterwards, I hosted the website on a server and tested it for errors. The bulletin board website successfully ran on the server and functioned correctly.
